Shell Commands
**************

.. note::
 
   Linux 系统中常见的3种命令行参数(以 :command:`ps` 为例):

      * Unix 类型的参数，前面有一短划线 如: ``ps -e``
      * BSD 类型的参数，前面没有短划线  如: ``ps  a``
      * GNU 长参数，前面有双划线 如: ``ps --deselect``
        
   所以，``ps –u`` 和 ``ps u`` 查询结果会不同.
